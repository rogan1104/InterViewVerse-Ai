# 🎤 InterviewVerse AI – Voice Analytics Module

## Overview

The Voice Analytics module is designed to evaluate a candidate's verbal communication during an AI interview.

It records interview audio, converts speech into text using OpenAI Whisper, and extracts useful communication metrics.

---

## Features

- Audio Recording
- Speech-to-Text
- Transcript Generation
- Word Count
- Speaking Rate (Words Per Minute)
- Filler Word Detection

---

## Tech Stack

- Python
- OpenAI Whisper
- SoundDevice
- SciPy
- FFmpeg

---

## AI Technologies Used

- OpenAI Whisper

No custom speech recognition model was trained.

---

## Output Metrics

- Transcript
- Total Words
- Interview Duration
- Speaking Rate
- Filler Words
